A relative path, e.g. common/subprojects/sharedlibs/, that will be added to the library path so that the library will be loaded from "${libBasePath}/src/**/*.groovy", "${libBasePath}/vars/*.groovy", "${libBasePath}/vars/*.txt", "${libBasePath}/resources/". If blank, the library will be loaded from default location under project root of code repository "src/**/*.groovy", "vars/*.groovy", "vars/*.txt", "resources/".
